# 💰 Crypto CLI Tool

> Ferramenta simples em Python para consultar preços de criptomoedas diretamente no terminal.

---

## 🚀 Features

- 📊 Consultar preço de uma criptomoeda
- 🔁 Calcular par entre duas criptomoedas
- ⚡ Rápido e leve (feito para terminal)
- 🎨 Saída formatada com `rich`

---

## 🧠 Tecnologias

- Python 3
- requests
- rich

---

## 📦 Instalação

```bash
pip install requests rich
